## NodeMCU ESP-12E carrier board for DashSentinel external display/status module.

Features:
- Amica-style NodeMCU ESP-12E footprint, 22.86 mm / 0.9 in header spacing
- 4-pin I2C OLED connector: GND, VCC, SCL, SDA
- 3 status LEDs: green/yellow/red
- 2 push buttons
- Through-hole components for easy hand soldering
